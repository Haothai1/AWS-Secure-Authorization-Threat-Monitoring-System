# AWS Secure Authorization & Application Security using SNS

Serverless security monitoring architecture built on AWS that detects malicious activity in real time and delivers automated real-time security alerts using Amazon SNS.

---

## üìå Overview

This project implements a secure, event-driven AWS architecture that protects web applications using a layered security model. 

The system filters malicious traffic, monitors suspicious activity, and sends real-time alerts to administrators via email/SMS using Amazon SNS.

The architecture uses:

- AWS WAF (Web Application Firewall)
- CloudFront (CDN Layer)
- Amazon S3 (Static Hosting)
- IAM (Role-Based Access Control)
- GuardDuty (Threat Detection)
- CloudWatch (Monitoring & Logging)
- EventBridge (Event Routing)
- AWS Lambda (Serverless Processing)
- Amazon SNS (Real-Time Notifications)
